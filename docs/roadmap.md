# Roadmap

Este roadmap organiza as próximas evoluções do EPMartins Automation Lab em fases, priorizando primeiro qualidade de portfólio, depois integrações reais e, por fim, evolução do laboratório como produto.

## Fase 1 — Qualidade de portfólio

- Corrigir links internos e navegação entre documentos.
- Validar importação dos 3 workflows no n8n.
- Adicionar prints ou previews visuais dos workflows em `assets/`.
- Adicionar exemplos de execução via `curl`.
- Revisar todos os READMEs com foco em clareza para recrutadores.
- Garantir que nenhum arquivo contenha credenciais, tokens ou dados sensíveis.

## Fase 2 — Integrações reais

- Adicionar envio de resultados para Google Sheets.
- Adicionar notificações em Slack ou Discord.
- Criar integração com Gmail ou caixa de entrada simulada.
- Criar versão com Notion ou ferramenta de CRM.
- Adicionar tratamento de erro para payload inválido.
- Criar logs de execução para auditoria e acompanhamento.

## Fase 3 — Evolução como produto

- Criar dashboard simples para disparar testes dos workflows.
- Adicionar biblioteca de prompts por segmento de negócio.
- Criar estratégias de fallback entre modelos de IA.
- Adicionar métricas de qualidade das respostas.
- Criar versionamento dos workflows por release.
- Adicionar novos workflows voltados a marketplace, suporte, produtividade e backoffice.

## Possíveis novos workflows

- Gerador de posts para LinkedIn com base em temas de carreira.
- Monitor de concorrentes em marketplaces.
- Classificador de leads comerciais.
- Gerador de resposta para atendimento ao cliente.
- Resumo automático de reuniões.
- Conversor de mensagens em tarefas para Notion, Trello ou Jira.
